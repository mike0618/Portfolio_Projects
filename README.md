# Text-Morse Package

This is a Text to Morse and vise-versa console code converter with auto detection conversion direction

# Usage
Console auto
```
from morse_text_miked import morse_text

morse_text.convert()
```
Encoding text to morse
```
from morse_text_miked import morse_text

morse_text.encode('text_here')
```
Decoding from morse
```
from morse_text_miked import morse_text

morse_text.decode('- . -..- - ..--.- .... . .-. .')
```
